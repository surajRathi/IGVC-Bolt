# Bolt
#### Team Abhiyaan's ROS based robot model <br />
Getting Started
---------------
#### To launch robot in gazebo, run
```
roslaunch igvc_bolt gazebo.launch
```

#### To visualize simulaion in rviz, run
```
roslaunch igvc_bolt display.launch
```
